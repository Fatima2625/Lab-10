Support Vector Machine (SVM) Assignment

This project demonstrates the use of a Support Vector Machine (SVM) classifier on the famous Iris flower dataset using Python and Scikit-Learn.

Dataset

The Iris dataset contains 150 flower samples from three species:

Setosa
Versicolor
Virginica

Each sample includes:

Sepal length
Sepal width
Petal length
Petal width
Libraries Used
Pandas
NumPy
Matplotlib
Seaborn
Scikit-Learn
Project Steps
Load the Iris dataset using Seaborn.
Perform exploratory data analysis (EDA).
Visualize the dataset using:
Pairplot
KDE Plot
Split the data into training and testing sets.
Train an SVM classifier using SVC().
Evaluate the model using:
Confusion Matrix
Classification Report
Improve the model using GridSearchCV.
Model Performance

The SVM model achieved high classification accuracy on the Iris dataset with excellent precision and recall scores.
